# Mise à jour du dépôt GitHub `xavpac/xavpac`

## Méthode simple depuis le navigateur

1. Décompresser le dossier `xavpac-v7-github` sur le Mac.
2. Ouvrir le dépôt `https://github.com/xavpac/xavpac`.
3. Cliquer sur **Add file**, puis **Upload files**.
4. Glisser tout le contenu du dossier décompressé dans la zone GitHub.
5. Saisir le message : `XavPac V7 - nouvelle interface Aviation et Moyens nationaux`.
6. Cliquer sur **Commit changes**.
7. Vercel reconstruira automatiquement le site si le dépôt est déjà relié.

## Contrôle visuel

Ajouter `?demo=1` à l'adresse du site pour afficher l'aperçu visuel :

```text
https://xavpac.vercel.app/?demo=1
```

Le site indique clairement **Aperçu visuel**. L'adresse normale reste en données réelles uniquement.
