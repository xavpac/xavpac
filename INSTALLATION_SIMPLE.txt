XAVPAC SIMPLE — INSTALLATION

Cette version a été préparée à partir de votre projet actuel.

MODIFICATIONS PRINCIPALES
- GPS réel : aucun emplacement de secours présenté comme votre position.
- Punaise 📍 pour la position GPS.
- Aucune météo sur la carte Aviation.
- Aucune météo des villes voisines dans Aviation.
- Météo uniquement pour la ville de départ et la ville d’arrivée.
- Page Moyens nationaux avec la même présentation générale qu’Aviation.
- Projet vérifié avec TypeScript et avec la compilation Next.js de production.

INSTALLATION DANS LE PROJET ACTUEL
1. Décompressez le ZIP dans Téléchargements.
2. Dans le terminal, lancez :

   bash ~/Downloads/XavPac-simple/INSTALLER.command

3. Une sauvegarde est créée automatiquement dans Documents.
4. Quand le contrôle est terminé, lancez :

   cd ~/xavpac
   npm run dev

5. Ouvrez http://localhost:3000

MISE EN LIGNE APRÈS VALIDATION LOCALE
   cd ~/xavpac
   git add .
   git commit -m "XavPac simple : aviation et moyens nationaux"
   git push origin main

Vercel redéploiera automatiquement si le dépôt GitHub est déjà relié.
