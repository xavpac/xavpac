XAVPAC 6.1 — INSTALLATION AUTOMATIQUE

1. Décompressez le fichier Installateur-XavPac-6.1-Aviation-FlightWall.zip.
2. Double-cliquez sur « Installer XavPac.command ».
3. Si macOS bloque le fichier : clic droit > Ouvrir > Ouvrir.
4. Laissez la fenêtre Terminal ouverte jusqu’au message final.

L’installateur :
- sauvegarde le dossier xavpac existant ;
- conserve la liaison GitHub (.git) ;
- installe les fichiers XavPac 6.1 ;
- exécute npm install, le contrôle TypeScript et le build ;
- crée un commit et pousse vers GitHub lorsque le dépôt est connecté ;
- ouvre ensuite https://xavpac-one.vercel.app.
